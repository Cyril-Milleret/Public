# WolverineGPSSurival
This repository contains Rcode, NimbleCode and data to reproduce the OPSCR model analysis as desribed in 

Milleret C, Bischof R, Dupont P, Brøseth H, Odden J, Mattisson J. 2021 *GPS collars have an apparent positive effect on the survival of a large carnivore.* Biol. Lett.20210128.https://doi.org/10.1098/rsbl.2021.0128
